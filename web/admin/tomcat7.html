Tomcat7 installation notes

Migrating from Tomcat6 to Tomcat7 on our Cornell host.

Downloaded a recent binary from http://tomcat.apache.org/download-70.cgi 

curl http://www.poolsaboveground.com/apache/tomcat/tomcat-7/v7.0.52/bin/apache-tomcat-7.0.52.zip > tmp.zip

cd /usr/share unzup tmp.zip

sudo ln -s apache-tomcat-7.0.52 tomcat7
cd tomcat7/bin
./catalina.sh start

Running tomcat7 as service (as per the advice at http://www.davidghedini.com/pg/entry/install_tomcat_7_on_centos ):

Create /etc/init.d/tomcat7 

chkconfig --list tomcat6
chkconfig tomcat6 off
chkconfig --list tomcat6
chkconfig tomcat7 on
